# 综合练习：学生成绩管理系统

## 项目简介

本综合练习需要你运用本模块学到的所有知识，完成一个简单的学生成绩计算程序。

## 项目需求

### 功能描述

创建一个程序，计算学生的各科平均分和总分，并给出等级评定。

### 具体要求

1. **数据结构**：使用元组存储学生信息（姓名、语文分数、数学分数、英语分数）

2. **函数实现**：

   - `calculate_total(scores: (i32, i32, i32)) -> i32`
     - 计算三科总分

   - `calculate_average(scores: (i32, i32, i32)) -> f64`
     - 计算三科平均分

   - `get_grade(average: f64) -> &'static str`
     - 根据平均分评定等级
     - 90-100: "优秀"
     - 80-89: "良好"
     - 70-79: "中等"
     - 60-69: "及格"
     - < 60: "不及格"

   - `print_report(name: &str, scores: (i32, i32, i32))`
     - 打印完整的学生成绩报告

3. **输出格式示例**：

   ```
   学生成绩报告
   ============
   姓名: 张三
   语文: 85
   数学: 92
   英语: 88
   ------------
   总分: 265
   平均分: 88.33
   等级: 良好
   ```

## 测试用例

```rust
// 测试总分计算
assert_eq!(calculate_total((85, 92, 88)), 265);

// 测试平均分计算
assert_eq!(calculate_average((100, 100, 100)), 100.0);
assert_eq!(calculate_average((0, 0, 0)), 0.0);

// 测试等级评定
assert_eq!(get_grade(95.0), "优秀");
assert_eq!(get_grade(85.0), "良好");
assert_eq!(get_grade(75.0), "中等");
assert_eq!(get_grade(65.0), "及格");
assert_eq!(get_grade(55.0), "不及格");
```
