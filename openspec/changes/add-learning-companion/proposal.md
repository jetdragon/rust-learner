# 变更：添加学习伴侣应用

## Why

当前仅依靠手动填写 `进度.md` 追踪学习进度存在以下问题：
- 缺乏提醒机制，容易拖延
- 进度可视化不足，难以看到进步
- 缺乏智能练习和掌握程度分析
- 缺乏激励和反馈机制

需要一个学习伴侣应用来增强学习体验和持续性。

## What Changes

- 创建一个桌面应用（使用 Tauri 框架）
- 解析当前仓库结构，读取学习进度
- 提供系统通知定时提醒学习
- 可视化展示学习进度（进度条、图表）
- 智能出题练习并分析掌握程度
- 根据掌握程度（≥95%）解锁下一模块
- 提供鼓励和成就系统

## Impact

- 影响的规格：`learning-companion`（新能力）
- 新增目录：`learning-companion/` 桌面应用代码
- 修改文件：可选地增强 `进度.md` 结构以支持自动解析
- 独立于学习模块，作为辅助工具存在

## 技术选型

- **前端**：React + TypeScript
- **桌面框架**：Tauri（轻量、跨平台）
- **状态管理**：Zustand
- **UI 组件**：shadcn/ui
- **图表**：Recharts
- **系统通知**：Tauri 插件
