# enhance-tui-navigation: 增强 TUI 导航和交互

## 问题描述

1. **光标移动跳行问题** - 在仪表板选择模块时，按一下方向键光标跳两行
2. **更新进度界面功能不足** - 只能标记完成，缺少"打开学习"功能
3. **任务操作不灵活** - 现有 5 个任务类型，但每个任务只能标记完成

## 解决方案

### 1. 修复光标跳行问题

**原因分析**: `draw_dashboard` 函数使用手动添加 `">> "` 前缀的方式，但 `ListState` 选中项也有高亮机制，可能导致重复渲染。

**修复方案**: 统一使用 ratatui 的 `highlight_symbol` 机制，移除手动前缀。

### 2. 新增模块详情界面

新增 `AppState::ModuleDetail` 状态，用于显示模块详细信息和提供操作选项。

**功能**:
- 显示模块名称和描述
- 显示模块包含的文件（README、exercises、tests、自检清单）
- 每个任务类型显示状态和可执行操作
- 对于未完成的任务，提供"打开"和"标记完成"两个操作

### 3. 增强任务操作

每个任务类型支持以下操作：

| 任务类型 | 打开操作 | 标记完成 |
|---------|---------|---------|
| 概念学习 | 打开 README.md | 标记完成 |
| 代码示例 | 打开示例文件 | 标记完成 |
| 练习题 | 打开 exercises.md | 标记完成 |
| 综合练习 | 打开 tests 目录 | 标记完成 |
| 自检清单 | 打开自检清单.md | 标记完成 |
