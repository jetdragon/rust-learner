# Rust 学习伴侣 - Web 版

一个现代化的 Web 版 Rust 学习伴侣，使用暖色调设计风格。

## 技术栈

### 后端
- Rust + Axum (Web 框架)
- SQLite (数据库)
- Tokio (异步运行时)

### 前端
- React 18 + TypeScript
- Vite (构建工具)
- Tailwind CSS (样式)
- Axios (HTTP 客户端)

## 快速开始

### 后端

```bash
cd server
cargo run
```

服务器将在 `http://localhost:3000` 运行

### 前端

```bash
cd client
npm install
npm run dev
```

开发服务器将在 `http://localhost:5173` 运行

## 功能

- 📊 学习仪表板
- 📚 模块进度追踪
- ✅ 任务完成标记
- 📝 练习测试
- 🏆 成就系统
- 📤 数据导出
- 🎨 暖色调设计风格

## 设计

采用暖色调配色方案，营造温馨的学习氛围：
- 橙色 (#f97316) - 主色调
- 琥珀色 (#ea580c) - 强调色
- 暖灰色 (#7c2d12) - 文本色
- 浅暖色 (#fff7ed) - 背景色
