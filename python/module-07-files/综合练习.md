# 综合练习：个人信息管理器

## 项目描述

创建一个简单的个人信息管理器，综合运用本模块所学的基础知识：
- 变量和数据类型
- 数据结构（列表、字典、集合）
- 函数

## 项目要求

### 1. 数据模型

创建以下数据结构来管理个人信息：

**用户信息字典**：
```python
user_info = {
    "name": "张三",
    "age": 25,
    "email": "zhangsan@example.com",
    "address": "北京市朝阳区",
    "phone": "13800138000"
}
```

**技能列表**：
```python
skills = ["Python", "数据分析", "机器学习", "英语六级"]
```

**兴趣集合**（自动去重）：
```python
interests = {"编程", "阅读", "运动", "编程", "旅行"}  # 会自动去重
```

你的实现应该通过以下测试：

