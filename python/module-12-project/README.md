# 综合项目：命令行工具

## 项目简介

本模块是一个综合项目，将运用所学知识创建一个命令行工具。

## 项目描述

创建一个简单的命令行任务管理工具。

## 功能要求

1. 添加任务
2. 列出任务
3. 标记完成
4. 保存到文件

## 使用的技术

- 参数解析 (argparse 或 click)
- 数据序列化 (json)
- 文件操作
- 命令行界面

## 示例

```bash
python tool.py add "完成作业"
python tool.py list
python tool.py done 1
```
