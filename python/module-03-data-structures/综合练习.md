# 综合练习：学生成绩管理系统

## 项目描述

使用学到的数据结构知识，创建一个简单的学生成绩管理系统。

## 数据模型

使用字典列表存储学生信息：

```python
students = [
    {
        "id": 1,
        "name": "张三",
        "age": 20,
        "scores": {
            "math": 85,
            "english": 90,
            "science": 88
        }
    },
    # ... 更多学生
]
```

## 需要实现的函数

1. 添加学生 `add_student(students, student)`
2. 查找学生 `find_student(students, name)`
3. 计算平均分 `calculate_average(student, subject=None)`
4. 获取排名 `get_ranking(students, subject)`
5. 更新成绩 `update_score(students, student_id, subject, score)`
6. 删除学生 `remove_student(students, student_id)`

## 预期输出

```text
学生列表：
1. 张三 (数学: 85, 英语: 90, 理综: 87.67)
2. 李四 (数学: 92, 英语: 88, 理综: 90.00)
...

数学排名:
1. 李四 (92)
2. 王五 (90)
3. 张三 (85)
...
```
