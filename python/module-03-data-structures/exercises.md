# 练习题

本模块包含数据结构相关的练习。

## 练习 1：列表操作

**难度**：简单

编写一个函数 `merge_sorted_lists(list1, list2)`：
- 合并两个已排序列表
- 返回新的排序列表

## 练习 2：字典反转

**难度**：中等

编写一个函数 `reverse_dict(d)`：
- 返回新的字典，键值互换

## 练习 3：集合交集

**难度**：中等

编写一个函数 `set_intersection(set1, set2)`：
- 返回两个集合的交集

## 练习 4：列表去重

**难度**：中等

编写一个函数 `remove_duplicates(lst)`：
- 返回去重后的列表（保持顺序）
- 使用集合但不能直接转换返回

## 练习 5：字典排序

**难度**：中等

编写一个函数 `sort_dict_by_values(d)`：
- 返回按值排序的键列表

## 练习 6：嵌套结构

**难度**：困难

处理嵌套字典列表，实现查询和更新功能。
