# 模块与包

## 模块简介

本模块介绍Python的模块系统，包括import语句、包管理和命名空间。

## 学习目标

完成本模块后，你将能够：

- 导入标准库和第三方模块
- 创建和组织自己的模块
- 理解命名空间
- 掌握 `__init__.py` 的作用

## 核心概念

### 导入语句

```python
import math
from datetime import datetime
from collections import defaultdict
```

### 包管理

使用 `pip` 安装第三方包：

```bash
pip install numpy
pip install pandas
```

### `__init__.py`

初始化包，定义导入行为。

## 代码示例

位于 `examples/` 目录：
- `imports.py` - 导入演示
- `packages.py` - 包组织演示
